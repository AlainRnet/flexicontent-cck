OVERRIDING a template file of the module (e.g.):
  news.php,  carousel.php,  select.php,  default.php,  ETC

copy the above to your Joomla template (JPATH_THEMES is your Joomla template folder)
   JPATH_THEMES /modules/mod_flexicontent/tmpl/news.php
   JPATH_THEMES /modules/mod_flexicontent/tmpl/carousel.php
   JPATH_THEMES /modules/mod_flexicontent/tmpl/select.php
   JPATH_THEMES /modules/mod_flexicontent/tmpl/default.php

and override them with customized code




TO OVERRIDE MODULE HEADER AND FOOTER
  _header.php, _footer.php
  
copy the above to your Joomla (JPATH_THEMES is usually /templates/ folder) 
   JPATH_THEMES /modules/mod_flexicontent/tmpl/_header.php
   JPATH_THEMES /modules/mod_flexicontent/tmpl/_footer.php

and override them with customized code
